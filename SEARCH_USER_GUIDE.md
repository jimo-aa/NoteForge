## ⌨️ 键盘快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **Cmd+K** / **Ctrl+K** | 打开/关闭搜索 | 全局快捷键 |
| **↑** | 上一个结果 | 导航到上一个搜索结果 |
| **↓** | 下一个结果 | 导航到下一个搜索结果 |
| **Enter** | 打开选中笔记 | 确认选择 |
| **Esc** | 关闭搜索 | 退出搜索模式 |

## 💡 使用技巧

### 技巧 1：精准搜索

使用完整的单词或短语获得更精准的结果：

```
❌ 搜索：Rus    → 结果较多且精度低
✅ 搜索：Rust   → 结果精准

❌ 搜索：学      → 结果过多
✅ 搜索：学习    → 结果精准
```

### 技巧 2：快速导航

使用键盘比鼠标更快：

```
1. Cmd+K               打开搜索
2. 输入搜索词          自动搜索
3. ↓ ↓ ↓              快速导航
4. Enter              打开笔记
```

### 技巧 3：组合搜索

搜索多个关键词来精化结果：

```
搜索一：学习
结果：50 条

搜索二：学习 Rust
结果：8 条（更精准）
```

## ❌ 常见问题

### Q1: 如何在特定笔记中搜索？

A: 打开笔记后按 Cmd+K，搜索词会限制在该笔记内。

```
步骤：
1. 打开某个笔记
2. 按 Cmd+K
3. 输入关键词
4. 结果只显示该笔记内的匹配
```

### Q2: 如何导出搜索结果？

A: 目前不支持直接导出，但可以：
- 复制笔记 ID
- 逐个打开并导出

### Q3: 搜索是否支持正则表达式？

A: 暂不支持正则表达式，仅支持普通文本搜索。

### Q4: 搜索速度会变慢吗？

A: 不会。搜索使用本地索引，速度很快（通常 < 50ms）。

### Q5: 如何禁用 Cmd+K 快捷键？

A: 快捷键由系统全局处理，暂不支持自定义。

## 🔐 隐私和安全

### 搜索数据

- ✅ **完全本地化** - 所有搜索在本地进行
- ✅ **无服务器** - 不上传任何数据
- ✅ **隐私保护** - 不记录搜索历史
- ✅ **加密支持** - 支持搜索加密内容

### 数据保存

```
✓ 搜索索引 - 本地存储在 .noteforge/index/
✓ 笔记数据 - 本地存储在 .noteforge/noteforge.db
✓ 用户数据 - 永不上传
```
