## 二、V1.0 发布标准

### 2.1 质量门禁

```
✅ TypeScript: tsc --noEmit 零错误
✅ Rust: cargo clippy -- -D warnings 零警告
✅ Java: gradlew build 全部测试通过 (59/59, JaCoCo 60%+)
✅ Frontend: vitest run 全部测试通过 (15/15)
✅ Rust Core: cargo test 全部测试通过 (20/20)
✅ Vite Build: npm run build 成功
✅ Tauri Build: cargo build --release 编译通过
□ Lighthouse: 桌面端 ≥ 85 (Performance) / ≥ 90 (Accessibility) — 需浏览器环境
```

### 2.2 用户体验标准

- 首次启动 < 2s（冷启动）
- 笔记打开 < 100ms
- 全文搜索 < 200ms（1 万条笔记）
- 自动保存 < 500ms 写入确认
- 同步延迟 < 3s（同区域）
- 离线可用（无后端完全可工作）
- 界面支持中英文
